How these links were built (per Unicode guidance)

  • Private window. Open every link in a New Private / Incognito / InPrivate window to minimise search personalisation.
  • Category term added. Each query is focused with an electronics qualifier (e.g. electrical-switch, battery-cell) so unrelated hits (light switch, biology cell) are filtered out — see the “why” note under each phrase.
  • Multiword terms hyphenated. Spaces are replaced with hyphens for Google Search/Video (e.g. variable-resistor) so results require the whole phrase, not scattered words.
  • “elephant” comparator. Google Trends & Ngram links compare each term against elephant in the same URL; Google Search/Video use the baseline elephant screenshots below.
  • Widest range + category. Trends links use date=all (web) / all_2008 (image) and the Computers & Electronics category (cat=5); Ngram uses 1500–2019.
  • Result count. On Google Search/Video, click Tools — the result count appears at the far right. Screenshot it.
